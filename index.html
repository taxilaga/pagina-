Características de Word, Excel y Access

Microsoft Word

  • Procesador de textos líder en el mercado.
  • Permite crear, editar y dar formato a documentos.
  • Incluye herramientas de ortografía y gramática.
  • Permite insertar imágenes, tablas, gráficos y ecuaciones.
  • Ofrece plantillas prediseñadas para diversos tipos de documentos.
  • Colaboración en línea y comentarios en tiempo real.
Video: ¿Qué es Word? Audio: Introducción a Word

Microsoft Excel

  • Hoja de cálculo potente y versátil.
  • Permite realizar cálculos automáticos y crear fórmulas.
  • Genera gráficos y tablas dinámicas para analizar datos.
  • Incluye funciones de filtrado y ordenamiento de información.
  • Facilita la gestión de grandes volúmenes de datos.
  • Permite colaboración en tiempo real y protección de hojas.
Video: Introducción a Excel Audio: ¿Qué es Excel?

Microsoft Access

  • Sistema de gestión de bases de datos relacional.
  • Permite crear y administrar bases de datos de manera sencilla.
  • Incluye herramientas para formularios, consultas e informes.
  • Facilita la automatización de tareas con macros.
  • Permite la integración con otras aplicaciones de Office.
  • Ideal para pequeñas y medianas empresas.
